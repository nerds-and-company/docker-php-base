# itmundi/php
Itmundi PHP image for Docker

## Contains

### OS
 - Based on the debian:jessie image

### Services
 - Apache 2.4
 - PHP 5.6

* Derived from [docker-libary/php](https://github.com/docker-library/php/tree/master/5.6/apache)

### PHP Extensions
 - mcrypt
 - mbstring
 - gd
 - pdo_mysql
 - zip
 - bz2

### Other
 - Contains git
 - Contains curl
 - Contains a MySQL client
 - Contains Redis CLI Tools
 - Contains composer
